## 完整原始需求

> 幫我分析市面上所有功能性益生菌以及乳酸菌的差異以及在臨床上需要有其權威性期刊的數據支撐，
> 文獻必須最近五年內，以及要有 PICO 的格式去處理每一個成分每一個菌株都需要至少五篇以上的
> 文獻支撐以及交叉比對都需要有註腳呈現的時候要出現英文跟全音之後縮寫後面可以使用縮寫沒有關係，
> 把市面上所有功能性的益生菌包含針對在免疫系統上面的輔助有哪些例如說 Beta 葡聚醣、益生元、後生元，
> 全部都要說明以及在臨床上搭配的藥物，例如說過敏藥旅行者腹瀉還有使用抗生素時應要注意哪些事項？
> 在功能性益生菌上面必須要講出睡眠、運動型、減脂、私密處、過敏、腸胃道保健、自體免疫者警語，
> 全部都要有潛在風險分析以及單方介紹完後協同複方的成分會達到哪一些效果，
> 每個成分劑量都需要有明確的劑量與使用時間方式，
> 分析完過後，幫我整理成一個網站，讓我能夠完整清楚明瞭所介紹的內容，
> 因為我後續還想要把我藥局所有益生菌的產品都做在上面，
> 讓我能夠按照客人的需求，或是想要詢問的菌株，讓我快速找到哪一個產品最適合他，
> 或是有哪幾個產品能夠給他做選擇。

## 需求拆解清單

### A. 內容要求

| # | 需求 | 規格 |
|---|------|------|
| A1 | 市面上所有功能性益生菌 / 乳酸菌分析 | 含菌株差異比較 |
| A2 | 每個菌株 / 成分 ≥ 5 篇文獻 | 近 5 年（2021–2026）|
| A3 | PICO 格式（P / I / C / O） | 每篇文獻均須有 |
| A4 | 交叉比對 | 不同菌株同一適應症比較 |
| A5 | 腳注格式 | 首次出現須完整英文全名 + 縮寫，後續可用縮寫 |
| A6 | 免疫輔助品 | Beta-葡聚醣（β-Glucan）、益生元（Prebiotics）、後生元（Postbiotics） |
| A7 | 藥物搭配說明 | 抗組織胺（Antihistamines）、旅遊者腹瀉（Traveler's Diarrhea）、抗生素（Antibiotics） |
| A8 | 功能分類 | 睡眠、運動型、減脂、私密處、過敏、腸胃道保健 |
| A9 | 自體免疫警語 | 含潛在風險分析 |
| A10 | 複方協同效果 | 單方 → 複方加成說明 |
| A11 | 明確劑量與使用時機 | 每個成分均須標示 |

### B. 網站功能要求

| # | 需求 | 規格 |
|---|------|------|
| B1 | 完整清楚呈現所有分析內容 | 可搜尋、可篩選 |
| B2 | 藥局產品管理 | 可新增藥局商品 |
| B3 | 客戶需求配對 | 依客人需求 → 推薦最適菌株 / 產品 |
| B4 | 快速查詢 | 依菌株或症狀快速找到對應產品 |
